  • remove foam from inside and reseat cables
  • open box and visually check it’s correct components
  • boot into bios to check you got correct components. Check RAM speed.
  • boot in to get windows key. Do not update.
  • then do fresh install and update fully
  • then remove windows bloatware using
    • winaero
    • bloatbox. See community link up top. Perhaps this tool is too aggressive?
  • install mb drivers
  • update bios
  • install graphic drivers
    • Make sure you are getting full refresh rate of monitor in advanced display adapter
    • inside nvidia set texture quality to high
  • make sure ram is running at full speed. Also called XMPP/DOHC
  • install apps
  • remove all apps from startup
  • remove unneeded services
  • go into individual apps and turn off internet features and auto updates, like foxit auto update, etc
